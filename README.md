# ShareLog

- [한글] 힘들거나 속마음을 편하게 털어놓고 싶을 때, 우리의 감정을 나누는 공간 <ShareLog>입니다.
- [English] <ShareLog> is a space where you can share your daily struggles or innermost thoughts and connect with others.

## 📝 프로젝트 소개 / Project Introduction

- [한글] ShareLog는 일상의 고민이나 속마음을 편하게 털어놓고, 서로의 감정에 공감하며 위로를 주고받는 일기 공유 서비스입니다. 혼자 앓던 마음을 밖으로 꺼내어 가벼워지고, 비슷한 고민을 가진 사람들과 이모지로 따뜻한 온기를 나눌 수 있습니다.
- [English] ShareLog is a diary-sharing service designed for users to comfortably express their daily worries or innermost thoughts and receive support through empathy. It helps you lighten your emotional burden and share warmth with others through emoji reactions.

## ✨ 주요 기능 / Key Features

- [한글] 일기 작성하기 / Write a Diary:\*\* 자신의 속마음을 자유롭게 기록할 수 있습니다.
- [English] You can freely record your inner thoughts and feelings.

- [한글] 이모지로 반응 남기기 / Emoji Reactions:\*\* 다른 사용자의 글에 이모지를 통해 간단하고 따뜻한 공감을 표현할 수 있습니다.
- [English] You can express your support and empathy for others' posts using simple and warm emojis.

## 🚀 실행 방법 / How to Run

- [한글] 프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따라주세요.
- [English]Follow these steps to run the project in your local environment:

```bash
git clone https://github.com/xlouisjungx/likelion_FE_my_blog.git
cd my-blog
npm install
npm run dev
```
