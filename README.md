# ShareLog

- 힘들거나 속마음을 편하게 털어놓고 싶을 때, 우리의 감정을 나누는 공간 <ShareLog>입니다.

## 📝 프로젝트 소개

- ShareLog는 일상의 고민이나 속마음을 편하게 털어놓고, 서로의 감정에 공감하며 위로를 주고받는 일기 공유 서비스입니다. 혼자 앓던 마음을 밖으로 꺼내어 가벼워지고, 비슷한 고민을 가진 사람들과 이모지로 따뜻한 온기를 나눌 수 있습니다.

## ✨ 주요 기능

- 일기 작성하기: 자신의 속마음을 자유롭게 기록할 수 있습니다.
- 이모지로 반응 남기기: 다른 사용자의 글에 이모지를 통해 간단하고 따뜻한 공감을 표현할 수 있습니다.

## 🚀 실행 방법

프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따라주세요.

저장소 복제

```bash
   git clone [레포지토리 주소]
   cd my-blog
   npm install
   npm run dev
```

## 보완점

- 현재는 주요 기능 중, 기본 기능안 개발되어 있어 지속적인 아이디어 회의를 통해 추가적인 개발을 이어나갈 생각입니다.
- 일기라는 민감한 부분인 만큼, 보완적인 부분에서도 개선해 나아갈 계획입니다.
